### 3、Q：为什么选择GitHub  
#### A：因为GitHub是一个神奇的工具，他能赋予我们程序员力量

## 如果您需要保持匿名，请注意如下事项：
### 1、在您提交前确保使用的GitHub账号昵称不包含个人实名信息
### 2、请设置您的GitHub账号不公开email地址
### 3、请确保上传的病例信息打码个人信息
### 4、git commit前使用git config -l命令查看当前项目配置，确保 user.name 和 user.email不包含个人信息
